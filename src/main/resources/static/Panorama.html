全景VR教学实验 教学流程

        什么是全景VR呢？全景又被称为360度全景或3D实景，是指将拍摄的水平方向360度，垂直方向180度的多张照片拼接成一张全景图像。通过后期漫游制作，让浏览者能用鼠标控制浏览的方向，可左可右、可上可下观看物体或场景，仿佛身临其境一般。全景VR漫游也是一种新兴的富媒体技术，可在制作中添加图片、声音、视频和超链接。全景VR技术优势明显，它使用最简单的方式能够全方位地展示360度球型范围内的所有景致，观者可以自由操纵任意角度互动性地观察场景，还可以添加视频、音乐、文本、超链接等多种媒体，内容丰富形式新颖，被广泛地应用于旅游、房地产等行业的各种场景的展示。720云是国内致力于全景技术应用的非常著名的云平台，其提供的全景编辑制作工具功能强大，操作简便，类似平台还有720think等平台。注意720云中所说的720全景实质上还是指360全景，即可上下左右旋转观看的全景，它是水平360°，垂直180°的全景图片，并不能叫720°全景，720云之所以这么命名720，其初衷是360度全景+360度服务。全景VR制作和播放的成本相对较低，相比三维制作，制作周期短，开发成本低，时效性强，而且对播放设备要求低，普通电脑手机均可播放。360全景图在VR作品中几乎随处可见，360全景也是我们走进虚拟现实技术领域的入门必经之路，也是我们扣响VR技术殿堂的敲门砖。
        全景VR课程实验教学主要分十二个步骤进行，包括：步骤01：学生体验VR全景作品魅力；步骤02：自主学习老师给定的慕课视频等教学材料；步骤03：课上翻转，检查学生自主学习效果；学生分组，接受具体实验任务；步骤05：素材准备；步骤06：素材加工_合成全景图；步骤07：在全景云平台展示全景；步骤08：作品比武，答辩评分；步骤09：撰写实验报告；步骤10：提出新问题：Unity中如何利用全景图创建天空盒；步骤11：在VR内容开发编辑器中应用全景图；步骤12：提出更有挑战性的问题供有兴趣有能力的同学去探索。

全景VR教学实验

 

实验目标与实验设备

  •  实验目的：

    • 学会使用单反/微单相机的拍摄全景素材的技巧；
    • 学会全景云台的安装、校准与使用；
    • 掌握素材拼接合成全景图的要求及技巧；
    • 掌握全景云平台编辑器的使用；
    • 初步掌握在Unity、Animater、及VR内容编辑器中应用全景图。

  •  实验设备（硬件与软件）：

    • HTC VIVE VR眼镜 / HTC Focus 一体机
    • 单反 / 微单相机 / 相机，鱼眼镜头、全景云台、三脚架；
    • 大疆无人机 “御”Mavic 2 专业版；
    • 软件：PTgui、Lightroom、Photoshop、Pano2VR、Kolor KolorEyes；

重点难点

 重点难点：

  • 重点：全景图的拼接与合成；
  • 难点：全景图的拼接与合成，学生作品质量意识的养成。
全景VR实验步骤01

 

步骤01：学生体验VR全景作品魅力

  •  实验步骤：

    • 学生使用VR设备体验全景VR作品案例效果及交互方式，感受VR作品效果和交互方式。如：萍乡博物馆全景VR作品（https://720yun.com/t/cbvknq8erib）。
    • 理解并掌握帧的概念、帧的类型以及帧的基本操作；
    • 理解掌握三种补间动画的区别与联系；

  •  实验设备：

    • HTC VIVE VR眼镜 / HTC Focus 一体机；

      图1 全景VR展示实验作品_萍乡博物馆全景VR展示

      在图2中，箭头符号为足迹互动方式，还可以作为注视互动效果，教师课上用HTC VIVE PRO设备在大屏中演示，学生可以使用HTC Focus一体机观看感受VR作品效果和交互方式，作品中可添加解说和背景音乐效果。
    • 扫码观看往期作品

                              

全景VR实验步骤02

 

步骤02：自主学习老师给定的慕课视频等教学材料

  •  实验步骤：

    步骤02：自主学习老师给定的慕课视频等教学材料
            教师提供原创教程及相关网络教程网址【720云全景系列教程 / 720think全景系列教程】，布置学生利用课余时间自主学习。本实验中，学生需要了解全景VR作品的形式与作用，要解决如何拍摄全景素材，如何处理素材合成全景图，以及如何利用720云等全景云平台编辑器制作全景作品的足迹交互等功能，以及作品发布等技能，要教会掌握学生充分利用网络自主学习的能力。

  •  学习要点：

    • 认识360全景；
    • 全景素材拍摄方法；
    • 全景素材拼接与合成技巧；
    • 全景互动在线编辑工具的使用；
全景VR实验步骤03

 

步骤03：课上翻转，检查学生自主学习效果

  •  实验步骤：

    步骤03：课上翻转，检查学生自主学习效果
            教师在课上以问答形式检查学生自主学习情况，并讲述的项目制作步骤、制作规范与要求，强调易出错的地方，强调作品的质量意识。

  •  解决问题：

    • 什么是360度全景？
    • 全景的特点与优势？
    • 全景图的应用领域？
    • 制作全景VR作品的流程？
    • 全景素材照片拍摄技巧；
    • 全景素材的拼接与合成技巧；
    • 全景系统编辑器工具使用；
    • 全景作品的上传与发布。
全景VR实验步骤04

 

步骤04：学生分组，确定具体实验任务

  •  实验步骤：

            学生分组，确定具体任务。分组讨论作品设计方案及交互方式。

  •  分组方法：

    • 每组2~4人；
    • 各组学生安排按自主组合+教师微调方式；
    • 全景课程实验场景尽量选校内或学校周边场景进行；
    • 各组全景制作任务各不相同；
全景VR实验步骤05

 

步骤05：全景拍摄素材准备

  •  实验步骤：

            学生利用课后时间使用脚架、专用云台、相机或手机拍摄素材图，注意单张全景需拍摄的数量、间隔角度和图片保存格式；演示无人机全景拍摄方法，需无人机拍摄的素材由班级统一安排人员操作。

    图2 使用全景云台拍摄素材

  •  拍摄技巧：

    温馨提示：
            拍摄时合成单张全景图所需的照片的数量以及相邻照片拍摄所间隔的角度，拍摄时选择合适的相机设置参数选择合适的曝光。拍摄时保存的图片格式选择RAW，后期进行图片拼接时，可将同一组内的图片导入到Lightroom内进行统一的调色处理，并将处理好的图片导出为JPG格式。
            附件：零基础学全景课程拍摄及航拍教程
    720专业版云台视频教程【https://pan.baidu.com/s/1AHKMCAk_yXNVu8w7EynwyQ 提取码：kvei 】

全景VR实验步骤06

 

步骤06：素材加工_合成全景图

  •  实验步骤：

            学生按组别分别使用PTgui、Lightroom、Photoshop、Pano2VR等软件处理素材图片，将素材图加工为质量较好的全景图。此步骤必须强调作品的质量意识，也是教学的重难点。图片拼接建议使用PTgui时，要掌握图片拼接的原理，并在最后输出为全景图之前要认真检查图片长宽比是否为2：1，输出的全景图中是否存在瑕疵，若存在瑕疵需掌握如何解决修复瑕疵的方法。在输出之前要对要输出的全景图做好相应的垂直线，确保输出的全景图的空间位置上关系正确。在处理无人机所拍摄的全景图时，需要将PTgui输出的全景图拖入到Photoshop中进行“补天“处理，需要掌握这个过程的原理步骤及技巧和如何去避免补天过程中“接缝”的出现，以及出现了“接缝”该如何去解决处理。

  •  学习材料：

    • 原创全景图拼接合成教程
    • 720云PTgui拼接示范教程
    • 全景图拼接效果对比
               
全景VR实验步骤07

 

步骤07：在全景云平台展示全景

  •  实验步骤：

    • 引导学生将创建好的全景图导入720云或720think等全景云平台中；
    • 使用全景云平台提供的编辑工具为全景作品添加各种交互方式；
    • 使用全景云平台提供的编辑工具为全景作品添加说明文字或配音配乐等完善作品功能；
    • 欣赏全景云平台中的商业级作品，找出特色明显的优秀作品，引导学生分析作品特点，研究制作方法。

    图3 720云全景系统制作工具

  •  学习材料：

    • 720云全景互动工具使用教程
    • 720think全景互动工具使用教程
全景VR实验步骤 08

 

步骤08：作品比武，答辩评分

  •  实验步骤：

            在课堂上各小组以上台答辩方式展示自己的全景作品，简要总结自己任务完成后的心得体会与不足，全班同学利用超星学习通APP实现全员参与互动评分。

全景VR实验步骤 09

 

步骤09：撰写实验报告

  •  实验步骤：

            撰写实验报告，总结实验成败经验教训。

全景VR实验步骤11

 

步骤11：在VR内容开发编辑器中应用全景图

  •  实验步骤：

            实验课上将自己创建全景图素材导入到VR内容开发编辑器V-Lions及VMake中，实现简单切换交互，并再次使用VR设备观看自己的作品，体验自己创作的交互方式，学生成就感倍增，学习热情自然高涨，也增加外接全景项目的自信心。

  •  学习材料：

    • V-Maker 虚拟现实课件编辑平台VMake帮助文件
    • V-Lions虚拟现实内容编辑工具帮助文件
全景VR实验步骤12

 

步骤12：提出更有挑战性的问题供有兴趣有能力的同学去探索

  •  实验步骤：

            假设不使用720云等全景平台，有没有其他方法也能在浏览器浏览全景图。

  •  学习材料：

    • threejs中文网
    • 实时的3D可视化工具Lumion
